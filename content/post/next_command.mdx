# 📁기본 명령어

## 📄 Shell 명령어
```shell
# 프로젝트 생성
$ npx create-next-app@latest 프로젝트명

# 서버 실행
$ npm run dev

# 서버 종료
$ ctrl+C

# 서버 종료
$ ctrl+C
```
<br />

## 📄 vsCode 명령어
```shell
# 기본 컴포넌트 생성
$ rafce

# 서버 실행
$ npm run dev

# 서버 종료
$ ctrl+C

# 서버 종료
$ ctrl+C
```
