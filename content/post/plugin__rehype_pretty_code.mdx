# 📁코드블럭 기능설정

## 📄 줄 번호 표시

```md
\`\`\`js showLineNumbers
// 코드 블럭에 data-line-numbers 속성 추가
\`\`\`
```

## 📄 시작 줄 번호 지정

```md
\`\`\`js showLineNumbers{3}
// 시작 줄이 {number}부터 번호가 생김
\`\`\`
```

## 📄 줄 단위 하이라이트

```md
\`\`\`js {1, 2-3, 5}
// 선택한 줄 하이라이트
\`\`\`
```

## 📄 제목 추가

```md
\`\`\`js title="제목입니다."
// 코드블럭에 제목 추가
\`\`\`
```
