# 기본 타입 선언

## 기본형

```js
// 문자열
cosnt str: string = 'Hello';
// 숫자
cosnt num: number = 208;
// 불리언
cosnt isTrue: boolean = true;
```

## 배열

```js showLineNumbers
let arr1: number[] = [10, 20, 30];
let arr2: Array<number> = [10 , 20, 30];
let arr3: Array<string> = ["hello", "world"];
let arr4: [string, number] = ["jinyoung", 24];
```

## 객체

```js title="test.tsx"
let gildong: object = { name: "gildong", age: 24 };
let person: { name: string; age: number } = {
  name: "gildong",
  age: 24
};
```

# 함수 타입 선언

## 함수 선언

함수의 타입을 선언할 때에는 함수의 `매개변수`와 `return`값의 타입을 선언한다.

```js
function doIt(a: string, b: string): string{
  return a + b;
}
```

추가로 함수 매개변수나 객체의 키값 유무가 불확실한 경우 `선택적 매개변수`를  
사용하여 해당 매개변수가 존재해도 되고 안해도 되게끔 만들어 줄 수 있다.

```js
// optional parameters(선택적 매개변수)
function doIt(a: string, b: string, c?: number): void{
  cosnole.log(a + b);
}

doIt('Hello', 'World');
```
